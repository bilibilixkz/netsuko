Netsuko 是一款为 Typecho 设计的博客主题，使用Tailwind CSS构建。

Netsuko 提供优雅的现代界面和便利的预留功能。Netsuko 原生支持日夜模式，PJAX无刷新与评论邮件提醒等实用功能，快速上手直接使用预留的数十个链接槽位，自带画廊/链接/时间轴等多个预设页面。

*自1.2.0版本起，Netsuko 已进入稳定阶段，除大量新功能，修复漏洞和稳定性优化以外将不会频繁推送版本更新。*

---
### 安装
1. 下载Release至你的 Typecho 主题目录并解压，建议将文件夹重命名为 netsuko。
2. 登录 Typecho 后台，在“外观”设置中启用 Netsuko。

### 注意事项

1.本项目基于 MIT License 开源。无论如何使用，您都必须在页脚保留 Netsuko 的链接和名称信息。
2.本主题运行时可选是否依赖 Tailwind CDN。
3.进入“后台 -> 外观 -> 设置外观”，你可以自定义许多的标签项，配置各项功能和样式。
4.Netsuko 在开发过程中在编辑器中使用了AI辅助。
