Netsuko 是一款为 Typecho 设计的博客主题，使用了Tailwind CSS构建。
Netsuko 提供优雅的现代界面和便利的预留功能。快速上手直接使用预留的数十个链接槽位，自带画廊/链接/时间轴等多个预设页面。

---
### 安装
1. 下载Release至你的 Typecho 主题目录并解压，建议将文件夹重命名为 netsuko。
2. 登录 Typecho 后台，在“外观”设置中启用 Netsuko。

### 注意事项

1.本主题目前通过 CDN 加载 Tailwind CSS。运行时，请确保环境网络畅通。后续会将CSS精简在本地。
2.进入“后台 -> 外观 -> 设置外观”，你可以自定义许多的标签项。
3.本项目基于 MIT License 开源。
4.Netsuko 在开发过程中在编辑器中使用了AI辅助。
